# Виселица (Gallows) 🎮

Консольная реализация классической игры "Виселица" на русском языке. Игрок должен угадать загаданное слово, вводя буквы по одной. За каждую неверную попытку рисуется элемент виселицы.

## Особенности ✨

- 🎯 Случайный выбор слова из более чем 100 русских слов
- 🎨 Визуальное отображение виселицы в процессе игры
- ✅ Проверка ввода (только русские буквы)
- 📊 Отображение оставшихся попыток и неверных букв
- 🏆 Поздравление при победе и сообщение о проигрыше
